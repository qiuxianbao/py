# PyCharm 快捷键

| 功能描述     | 快捷键组合                |
|----------|----------------------|
| 删除       | `Ctrl + X`           |
| 复制       | `Ctrl + D`           |
| 最近的文件    | `Ctrl + E`           |
| 查看当前类的方法 | `Ctrl + F12`         |
| 关闭标签     | `Ctrl + F4`          |
| 重命名      | `Shift + F6`         |
| 查找文件     | `Shift`双击            |
| 上下移动     | `alt + Shift + 上下箭头` |
| 查找文本     | `Ctrl + Shift + R`   |
| 切换大小写    | `Ctrl + Shift + U`   |
| 返回上一个位置  | `Ctrl + alt + 左右箭头`  |
| 生成变量     | `Ctrl + alt + V`     |
| 格式化代码    | `Ctrl + alt + L`     |
| 删除无用的包   | `Ctrl + alt + O`     |
| 具体实现     | `Ctrl + alt + B`     |
